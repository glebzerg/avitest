## Общая информация

**Base URL:** `https://qa-internship.avito.com`  
**Формат данных:** JSON  
**Аутентификация:** Не требуется (тестовый стенд)

### Структура объявления

```json
{
  "id": "string",
  "sellerId": "integer",
  "name": "string",
  "price": "integer",
  "statistics": {
    "likes": "integer",
    "viewCount": "integer",
    "contacts": "integer"
  },
  "createdAt": "string (ISO 8601)"
}
```

### Техники тест-дизайна

- **Эквивалентное разбиение** — для валидации полей
- **Анализ граничных значений** — для числовых полей
- **Таблица принятия решений** — для комбинаций параметров
- **Переход состояний** — для жизненного цикла объявления
- **Предугадывание ошибок** — для негативных сценариев
